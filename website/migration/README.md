# Migration Guides

This directory provides migration paths from common state libraries to `scope-flux`.

Read this section if you already have a production store and want to migrate gradually with low risk.

## Start Here

1. Read `index.md` for common migration strategy.
2. Choose your current library guide.
3. Migrate one feature at a time, not the full app at once.

- `zustand.md`
- `jotai.md`
- `valtio-mobx.md`
